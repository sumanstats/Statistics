# Descriptive statistics



These are the statistics that describe the given data. 

The following are basic functions expected of a Statistics package:

+ `mean`
+ `median`
+ `mode`
+ `percentile`
+ `range`
+ `IQR` (interquartile range)
+ `mad`
+ `fivenum`
+ `sd`
+ `coef_variation`

Ther are available with 

```perl
use Statistics;
```
